Profiles Library

Το Profiles Library είναι ένα δυναμικού περιεχομένου πρόγραμμα το οποίο μπορείτε να ενσωματώσετε στην ιστοσελίδα σας. Το πρόγραμμα αυτό μπορεί να κάνει τα έξεις: Διαχείριση και εμφάνιση προφίλ σε ιστοσελίδες, ένα από τα καλύτερά προφίλ που υπάρχουν στο διαδίκτυο, μπορείτε ακόμη να κάνετε ιδιωτική κάποια σελίδα, υποστηρίζει status σε εμφάνισή blog, σελίδα με εικόνες, περιγραφή, και υποστηρίζει πολλές αθλητικές λειτουργείς, που ελάχιστα προφίλ κάνουν παγκόσμια, Τα προφίλ υποστηρίζουν κανονικά seo και δεν απαιτούν καμία προσθήκη από την μεριά του χρήστη π.χ. Flash και δεν απαιτούν καμία προσθήκη από την μεριά του server, π.χ. όχι-Windows-Asp, λειτουργούν μόνο με PHP,HTML,CSS,JS.

  • Page 1, Εμφανίζει ειδήσεις, εικόνες, βίντεο, σε μορφή blog ή αλλιώς χρονολογικό
  • Page 2, Γενική περιγραφή του χρήστη
  • Page 3, Εμφανίζετε διάφορές φωτογραφίες χωρισμένες σε κατηγορίες
  • Page 4, Εμφανίζει τα todaytips για αθλητικές ιστοσελίδες.
  • Page 5, Εμφανίζει τα records για αθλητικές ιστοσελίδες, και φτιάχνει μία σύνοψη για όλα έχασε ή κέρδισε συνολικά, ανά παιχνίδι, καθώς και υπολογισμό χρημάτων
  • Admin Page 1, Ανεβάζεται ειδήσεις, εικόνες, βίντεο, σε μορφή blog ή αλλιώς χρονολογικό
  • Admin Page 2, Επεξεργασία τις γενική περιγραφή του χρήστη
  • Admin Page 3, Εμφανίζετε από το διαδίκτυο διάφορες φωτογραφίες χωρισμένες σε κατηγορίες
  • Admin Page 4, Ανεβάζεται φωτογραφίες από τον υπολογιστή σας και της βάζετε στης κατηγορίες ή στο προφίλ ή στον φόντο, διαχειρίζεστε την σελίδα ιδιωτικής χρήσης
  • Admin Page 5, Βάζετε τα Today tips κ' όμως υποστηρίζει και παρολί παιχνίδια
  • Admin Page 6, Επεξεργάζεστε τα today tips
  • Admin Page 7, Βάζετε τα Today tips στα records page
  • Admin Options 1, Διαγραφή οτιδήποτε με απλό κλικ, on/off
  • Admin Options 2, Αλλάζει την ημερομηνία δημοσίευσης, on/off

Στο μέλλον πέρα από την αθλητικές δυνατότητες θα μπουν δυνατότητες social, chat και επικοινωνίας των χρηστών μεταξύ τους, likes, σχόλια κ.τ.λ, ποστ σε προφίλ άλλων κ.τ.λ.

Σε λίγες μέρες τα δημοσιευτεί η πρώτη υπό ανάπτυξη έκδοσή του προγράμματος εδώ στο github, ενώ σύντομα θα βγει η τελική έκδοση, θα ακολουθήσουν plugins για joomla, wordpress, getsimple ενώ αναμένονται και άλλα μετά, θα είναι δωρεάν και ανοιχτού κώδικα.

Παραδείγματα ενσωμάτωσης με PHP

Για αυτούς που έχουν πρόσβαση στα Server Side Scripts (PHP), μπορούν να προσθέσουν αυτό στην υπάρχων σελίδα τους, είναι και το προτεινόμενο.

<div style="width=1200 height="1200" >
<? 
$base = "user1";
$options = "nodates";
include "libandreas/gui-notes.php"; 
?>
</div>

Παραδείγματα ενσωμάτωσης με HTML

Για αυτούς που λειτουργούν ένα CMS σύστημα διαχείρισης περιεχομένου ίσως να είναι ποιο εύκολο να χρησιμοποιήσουν απλός ένα ενσωματωμένο πλαίσιο στην σελίδα τους, με HTML.

<iframe align="top" 
frameborder="0" 
height="1200" 
scrolling="no" 
src="libandreas/gui-notes.php?base=user1&options=nodates" 
width="780">
</iframe>
