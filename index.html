Andreas Source

Καλωσορίσατε στο ανοιχτού κώδικα αποθετήριο του Andreas Source.

Εδώ θα βρείτε όλα τα ανοιχτά project τα οποία είναι επί το πλείστον προγράμματα για την λειτουργία και κατασκευή δυναμικών ιστοσελίδων ειδικών λειτουργιών, τα project έχουν σαν σκοπό να χρησιμοποιηθούν από τρίτους κατασκευαστές ιστοσελίδων, η οποίοι χρησιμοποιήσουν τα project μου, και να φτιάξουν έτσι εύκολα την δική τους ιστοσελίδα χωρίς πολλές γνώσεις και χωρίς πολύ χρόνο.

Προγράμματα-CMS-Plugins

Είναι ωραία να φτιάχνεις την δική σου ιστοσελίδα χρησιμοποιώντας έτοιμα CMS όπως joomla, wordpress, κ.τ.λ. βρίσκεις ένα καλό template το φτιάχνεις στα μετρά σου, βάζεις το περιεχόμενο και όλα ΟΚ. Τι γίνετε όμως όταν θέλεις να κάνεις μια πρόσθετη λειτουργία στην Ιστοσελίδα? τότε θα πρέπει να αναζητήσεις plugins-πρόσθετα τα όποια θα πρέπει να εγκαταστήσεις και να ρυθμίσεις, σε αυτήν την ενότητα δεν θα δείτε τα site που έχω φτιάξει τα οποία είναι στην επόμενη ενότητα, αλλά θα δείτε προγράμματα που έχουν φτιαχτεί από το μηδέν για να στηρίξουν μία ιστοσελίδα, θα δείτε δλδ εργαλεία για κατασκευή ιστοσελίδων ειδικών λειτουργιών.

Profiles Library

The Profiles Library is a dynamic concent library, that you can include on your php-build website This Library-Plugin does this for you: Managment and show profiles in your website, meaby one of the most nice profiles outthere for websites, you can make your public or even private profile, something like a mini-blog or a mini-profile that will included on the websites.

  • Page 1, You can publish news and photos and videos on a timeline style blog page.
  • Page 2, You can publish a generic info about you.
  • Page 3, You can publish some photos in categories on the profiles
  • Page 4, You can publish Today Tips in a table for Sport websites.
  • Page 5, You can publish Your Records in a table for Sport websites, and make a table-style synoptis with you current amount and with all your lose-win-draw games.
  • Admin Page 1, Edit your blog or timeline profile
  • Admin Page 2, Edit you info, and put profile and background images, even edit them!
  • Admin Page 3, Edit and put your photos in categories.
  • Admin Page 4, Upload your photos profile or background or in categories.
  • and make a page privacy if you want and control who user can see it or not
  • Admin Page 5, Put your today tips
  • Admin Page 6, Edit your today tips
  • Admin Page 7, Put today tips on the records page
  • Admin Options 1, Delete anything on the profile with one link, on/off
  • Admin Options 2, Change upload date of anything on the profile with one link, on/off

Παραδείγματα ενσωμάτωσης με PHP

Για αυτούς που έχουν πρόσβαση στα Server Side Scripts (PHP), μπορούν να προσθέσουν αυτό στην υπάρχων σελίδα τους, είναι και το προτεινόμενο.

<div style="width=1200 height="1200" >
<? 
$base = "user1";
$options = "nodates";
include "libandreas/gui-notes.php"; 
?>
</div>

Παραδείγματα ενσωμάτωσης με HTML

Για αυτούς που λειτουργούν ένα CMS σύστημα διαχείρισης περιεχομένου ίσως να είναι ποιο εύκολο να χρησιμοποιήσουν απλός ένα ενσωματωμένο πλαίσιο στην σελίδα τους, με HTML.

<iframe align="top" 
frameborder="0" 
height="1200" 
scrolling="no" 
src="libandreas/gui-notes.php?base=user1&options=nodates" 
width="780">
</iframe>

Ιστοσελίδες που έχουν κατασκευαστεί

Etergat.gr

Πρόκειται για ένα απλό με 5-6 σελίδες και μια σελίδα επικοινωνίας με χάρτη, πλήρως δυναμικό.

LaxamentumRes.eu

Πρόκειται για site με πολύπλοκο template, αρκετά πολύπλοκο καθώς δεν είχε καμία σύστημά περιεχομένου και όλο το κείμενο στην html ήταν συνδεόμενο με js και css παντού, διαθέτη και έναν αυτόνομο σύστημά επικοινωνίας.

SharkBet.gr - Υπό κατασκευή

Πρόκειται για ένα από τα ποιο επαγγελματικά site που έχουν φτιαχτεί ποτέ για το ελληνόφωνους επισκέπτες, είναι φτιαγμένο χωρίς CMS, και είναι παρόλα αυτά πλήρως δυναμικό. Αναλαμβάνει πολύπλοκες ειδικές λειτουργίες που τα joomla, wordpress, drupal, getsimple, δεν κάνουν και ούτε είναι η δουλειά τους να κάνουν, για αυτό την δουλεία του δυναμικού την αναλαμβάνουν custom php κώδικας, οποίος βρίσκετε μέσα στην html σελίδες αλλά και σε ξεχωριστά αρχεία, δεν διαθέτει βάση δεδομένων τύπου .db γιατί οι πληροφορίες αποθηκεύονται απευθείας στο σύστημα αρχείων, μεταφέρετε από server σε server πολύ εύκολα με μία απλή αντιγραφή αρχείων, είναι τελείως αυτόνομο λειτουργεί σε όλους τους server.
Πρόκειται για ένα στοιχηματικό - αθλητικό site, φτιαγμένο πλήρως από το μηδέν, περιλαμβάνει και τα προφίλ, περιλαμβάνει custom (όχι έτοιμα) σύστημα πληρωμών, σύστημα έγγραφης και διαχείρισής χρηστών, σύστημα διαχείρισής παραγγελιών και συνδρομών και ξεκλείδωμα σελίδων, και ειδικό πρόγραμμα tv, πολύπλοκα μενού με εικόνες με css και JQuery (JavaScript), καθώς και ειδήσεις με ημερολόγιο αναζήτησης, στο μέλλον θα αποκτήσει online radio ή tv.
