Andreas CMS

Andreas CMS or A-c.ms is a ready to use e-Commerce platform for your server,
Plug & Play, clever, service independable & fully customizable.

Πρόκειται μια μεγάλη εφαρμογή αναζήτησης προϊόντων, συνεργάζεται με πολλά καταστήματα και υποκαταστήματα της επιλογή σας, όπου με βάση την ρoή feed του καταστήματος, συλλέγονται όλα τα προϊόντα από όλα τα συνεργαζόμενα e-shop, και εμφανίζονται με βάση τη χαμηλότερη τιμή, ή με βάση την κοντινότερη απόσταση, έπεισης επιτρέπει την χειροκίνητη προσθήκη των προϊόντων για τα μεμονωμένα καταστήματα.

Tablet Application

Ηλεκτρονικός Κατάλογος (e-Catalog), μεταξύ άλλων περιλαμβάνει, έτοιμο html που μπορεί να προσαρμοσθεί άμεσα σε οποιαδήποτε πελάτη, περιλαμβάνει, slider, λίστα προϊόντων και μενού.

Ο Κώδικας είναι της μορφής HTML για τα γραφικά.. αλλά και πάλι δεν είναι ο ίδιος Κώδικας html/responsive που φτιάχνω στα site, έχω περάσει ειδική διάσταση (vertical units) σε οτιδήποτε εμφανίζετε στην εφαρμογή, ώστε να προσαρμόζετε αυτόματα σε όλες τς οθόνες, αυτό σημαίνει ότι δεν αλλάζει διάταξη η μειώνει κενά σε όλες τις οθόνες, αλλά μένει ακριβώς ίδιο (auto-fit) σε όλες τις οθόνες, ακόμη και το κείμενο αλλάζει πάντα στο ίδιο σημείο σειρά, σαν να είναι π.χ. μια εικόνα, ενωθείτε πως δεν είναι εικόνα, αλλά πλήρως επιλέξιμο και διαδραστικό.

  • Γραφικά: HTML, Javascript, CSS, +Special Application Filters, +Vertical Units.
  • Σύστημα: Java, +Android Api Level 24.
