    WebSite Libraries

    Είναι ωραία να φτιάχνεις την δική σου ιστοσελίδα χρησιμοποιώντας έτοιμα CMS όπως joomla, wordpress, κ.τ.λ. βρίσκεις ένα καλό template το φτιάχνεις στα μετρά σου, βάζεις το περιεχόμενο και όλα ΟΚ. Τι γίνετε όμως όταν θέλεις να κάνεις μια πρόσθετη λειτουργία στην Ιστοσελίδα? τότε θα πρέπει να αναζητήσεις plugins-πρόσθετα τα όποια θα πρέπει να εγκαταστήσεις και να ρυθμίσεις, σε αυτήν την ενότητα δεν θα δείτε τα site που έχω φτιάξει τα οποία είναι στην επόμενη ενότητα, αλλά θα δείτε προγράμματα που έχουν φτιαχτεί από το μηδέν για να στηρίξουν μία ιστοσελίδα, θα δείτε δλδ εργαλεία για κατασκευή ιστοσελίδων ειδικών λειτουργιών.

    Heavy Projects

    SharkBet.gr

    Πρόκειται για ένα από τα ποιο επαγγελματικά site που έχουν φτιαχτεί ποτέ για το ελληνόφωνους επισκέπτες, είναι φτιαγμένο χωρίς CMS, και είναι παρόλα αυτά πλήρως δυναμικό. Αναλαμβάνει πολύπλοκες ειδικές λειτουργίες που τα joomla, wordpress, drupal, getsimple, δεν κάνουν και ούτε είναι η δουλειά τους να κάνουν, για αυτό την δουλεία του δυναμικού την αναλαμβάνουν custom php κώδικας, οποίος βρίσκετε μέσα στην html σελίδες αλλά και σε ξεχωριστά αρχεία, δεν διαθέτει βάση δεδομένων τύπου .db γιατί οι πληροφορίες αποθηκεύονται απευθείας στο σύστημα αρχείων, μεταφέρετε από server σε server πολύ εύκολα με μία απλή αντιγραφή αρχείων, είναι τελείως αυτόνομο λειτουργεί σε όλους τους server.
    Πρόκειται για ένα στοιχηματικό - αθλητικό site, φτιαγμένο πλήρως από το μηδέν, περιλαμβάνει και τα προφίλ, περιλαμβάνει custom (όχι έτοιμα) σύστημα πληρωμών, σύστημα έγγραφης και διαχείρισής χρηστών, σύστημα διαχείρισής παραγγελιών και συνδρομών και ξεκλείδωμα σελίδων, και ειδικό πρόγραμμα tv, πολύπλοκα μενού με εικόνες με css και JQuery (JavaScript), καθώς και ειδήσεις με ημερολόγιο αναζήτησης, στο μέλλον θα αποκτήσει online radio ή tv.

    Profiles Library

    Το Profiles Library είναι ένα δυναμικού περιεχομένου πρόγραμμα το οποίο μπορείτε να ενσωματώσετε στην ιστοσελίδα σας. Το πρόγραμμα αυτό μπορεί να κάνει τα έξεις: Διαχείριση και εμφάνιση προφίλ σε ιστοσελίδες, ένα από τα καλύτερά προφίλ που υπάρχουν στο διαδίκτυο, μπορείτε ακόμη να κάνετε ιδιωτική κάποια σελίδα, υποστηρίζει status σε εμφάνισή blog, σελίδα με εικόνες, περιγραφή, και υποστηρίζει πολλές αθλητικές λειτουργείς, που ελάχιστα προφίλ κάνουν παγκόσμια, Τα προφίλ υποστηρίζουν κανονικά seo και δεν απαιτούν καμία προσθήκη από την μεριά του χρήστη π.χ. Flash και δεν απαιτούν καμία προσθήκη από την μεριά του server, π.χ. Windows-Asp, λειτουργούν μόνο με PHP,HTML,CSS,JS.

    • Page 1, Εμφανίζει ειδήσεις, εικόνες, βίντεο, σε μορφή blog ή αλλιώς χρονολογικό
    • Page 2, Γενική περιγραφή του χρήστη
    • Page 3, Εμφανίζετε διάφορές φωτογραφίες χωρισμένες σε κατηγορίες
    • Page 4, Εμφανίζει τα todaytips για αθλητικές ιστοσελίδες.
    • Page 5, Εμφανίζει τα records για αθλητικές ιστοσελίδες, και φτιάχνει μία σύνοψη για όλα έχασε ή κέρδισε συνολικά, ανά παιχνίδι, καθώς και υπολογισμό χρημάτων
    • Admin Page 1, Ανεβάζεται ειδήσεις, εικόνες, βίντεο, σε μορφή blog ή αλλιώς χρονολογικό
    • Admin Page 2, Επεξεργασία τις γενική περιγραφή του χρήστη
    • Admin Page 3, Εμφανίζετε από το διαδίκτυο διάφορες φωτογραφίες χωρισμένες σε κατηγορίες
    • Admin Page 4, Ανεβάζεται φωτογραφίες από τον υπολογιστή σας και της βάζετε στης κατηγορίες ή στο προφίλ ή στον φόντο, διαχειρίζεστε την σελίδα ιδιωτικής χρήσης
    • Admin Page 5, Βάζετε τα Today tips κ' όμως υποστηρίζει και παρολί παιχνίδια
    • Admin Page 6, Επεξεργάζεστε τα today tips
    • Admin Page 7, Βάζετε τα Today tips στα records page
    • Admin Options 1, Διαγραφή οτιδήποτε με απλό κλικ, on/off
    • Admin Options 2, Αλλάζει την ημερομηνία δημοσίευσης, on/off

    Στο μέλλον πέρα από την αθλητικές δυνατότητες θα μπουν δυνατότητες social, chat και επικοινωνίας των χρηστών μεταξύ τους, likes, σχόλια κ.τ.λ, ποστ σε προφίλ άλλων κ.τ.λ.

    Σε λίγες μέρες τα δημοσιευτεί η πρώτη υπό ανάπτυξη έκδοσή του προγράμματος εδώ στο github, ενώ σύντομα θα βγει η τελική έκδοση, θα ακολουθήσουν plugins για joomla, wordpress, getsimple ενώ αναμένονται και άλλα μετά, θα είναι δωρεάν και ανοιχτού κώδικα.

    Παραδείγματα ενσωμάτωσης με PHP

    Για αυτούς που έχουν πρόσβαση στα Server Side Scripts (PHP), μπορούν να προσθέσουν αυτό στην υπάρχων σελίδα τους, είναι και το προτεινόμενο.

    <div style="width=1200 height="1200" >
<? 
$base = "user1";
$options = "nodates";
include "libandreas/gui-notes.php"; 
?>
</div>

    Παραδείγματα ενσωμάτωσης με HTML

    Για αυτούς που λειτουργούν ένα CMS σύστημα διαχείρισης περιεχομένου ίσως να είναι ποιο εύκολο να χρησιμοποιήσουν απλός ένα ενσωματωμένο πλαίσιο στην σελίδα τους, με HTML.

    <iframe align="top" 
frameborder="0" 
height="1200" 
scrolling="no" 
src="libandreas/gui-notes.php?base=user1&options=nodates" 
width="780">
</iframe>
