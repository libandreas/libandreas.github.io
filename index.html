Προγράμματα-CMS-Plugins

Είναι ωραία να φτιάχνεις την δική σου ιστοσελίδα χρησιμοποιώντας έτοιμα CMS όπως joomla, wordpress, κ.τ.λ. βρίσκεις ένα καλό template το φτιάχνεις στα μετρά σου, βάζεις το περιεχόμενο και όλα ΟΚ. Τι γίνετε όμως όταν θέλεις να κάνεις μια πρόσθετη λειτουργία στην Ιστοσελίδα? τότε θα πρέπει να αναζητήσεις plugins-πρόσθετα τα όποια θα πρέπει να εγκαταστήσεις και να ρυθμίσεις, σε αυτήν την ενότητα δεν θα δείτε τα site που έχω φτιάξει τα οποία είναι στην επόμενη ενότητα, αλλά θα δείτε προγράμματα που έχουν φτιαχτεί από το μηδέν για να στηρίξουν μία ιστοσελίδα, θα δείτε δλδ εργαλεία για κατασκευή ιστοσελίδων ειδικών λειτουργιών.

Profiles Library

The Profiles Library is a dynamic concent library, that you can include on your php-build website This Library-Plugin does this for you: Managment and show profiles in your website, meaby one of the most nice profiles outthere for websites, you can make your public or even private profile, something like a mini-blog or a mini-profile that will included on the websites.

  • Page 1, You can publish news and photos and videos on a timeline style blog page.
  • Page 2, You can publish a generic info about you.
  • Page 3, You can publish some photos in categories on the profiles
  • Page 4, You can publish Today Tips in a table for Sport websites.
  • Page 5, You can publish Your Records in a table for Sport websites, and make a table-style synoptis with you current amount and with all your lose-win-draw games.
  • Admin Page 1, Edit your blog or timeline profile
  • Admin Page 2, Edit you info, and put profile and background images, even edit them!
  • Admin Page 3, Edit and put your photos in categories.
  • Admin Page 4, Upload your photos profile or background or in categories.
  • and make a page privacy if you want and control who user can see it or not
  • Admin Page 5, Put your today tips
  • Admin Page 6, Edit your today tips
  • Admin Page 7, Put today tips on the records page
  • Admin Options 1, Delete anything on the profile with one link, on/off
  • Admin Options 2, Change upload date of anything on the profile with one link, on/off

Profiles Library Install

PHP INCLUDES METHOD

Just copy the folders and files in the root directory on in a folder if you want to include them in your site. for example if you can access php server side scripts include the page directly, for example:

<div style="width=1200 height="1200" >
<? 
$base = "user1";
$options = "nodates";
include "libandreas/gui-notes.php"; 
?>
</div>

HTML IFRAME METHOD

For those that do use a CMS driven website like joomla or wordpress, may use this method, because of the complicated way to edit the php files, or if you have a static webpage you can do this in html.

<iframe align="top" 
frameborder="0" 
height="1200" 
scrolling="no" 
src="libandreas/gui-notes.php?base=user1&options=nodates" 
width="780">
</iframe>
