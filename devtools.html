The Liquid UI

The Liquid UI is fit on screen dimensions, that mean with any device, big or small tablets or even desktop! The APP it looks really the same! As an image! Even text not goes on underline on different devices, Very good with Cordova or similar platform!

Specs:

Coding: HTML5 / CSS3 / Javascript 6

Use: "Specially design for Tablets"

Futures:

  • Native Material UI for Android
  • Swipe on slider to view next image
  • Touch on slider to view full screen
  • Separate CSS for the colors theme
  • Separate CSS for effects, shadows etc
  • Very light and fast
  • Auto position on 2 columns for all your description boxes.. Like fluid! No need to set position manually.
  • Full Liquid UI that fits on screen dimensions (vw & vh insdead of px, pt, etc) this makes interface to looks exactly same include text, on all devices.
  • Enough big UI for ease use and read
  • Code and files are very organized
  • Looks like a native android application, but written in HTML/CSS.
  • Explain code in comments, plus sample code
